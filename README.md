# React-BoilerPalte

## 적용 Stack

- Webpack(V5, ESM, babel, ts-node) + React
- Emotion (Styled)
- React-query + Axios
- yarn berry v4.0.2 (레포 생성 시점 버전)

## Started

```js

 // local start

 git clone https://github.com/yunwoo-yu/React-BoilerPlate.git

 cd React-BoilerPlate

 yarn

 yarn dev // start http://localhost:3000

```
